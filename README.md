# Smarthome Project
A custom smart-home built primarily with Python.

## Features
- [ ] Room entrance

  - [ ] Automatic *main* lighting (Wiz Smart Lights)
  - [ ] Automatic *feature* lighting (LED strips)
  - [ ] Speech (notifications, ?) with Amazon Alexa
    - [ ] Daily progress
    - [ ] Time read
- [ ] Remote control (web interface)
  - [ ] Control lights
  - [ ] Turn on alarm
    - [ ] Phone notifications
